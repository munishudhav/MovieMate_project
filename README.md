# MovieMate - Movie Search Web Application

## Project Overview

MovieMate is a movie search web application where users can search for movies and view movie details such as movie poster, title, and release year. The application fetches movie data from the OMDB API and displays movies dynamically on the website.

## Public API Used

This project uses the OMDB API (Open Movie Database API) to fetch movie data.
API Link: https://www.omdbapi.com/

## Planned Features

* Movie search functionality
* Display movie poster, title, and year
* Filter movies by year
* Sort movies by title
* Add movies to favorites
* Dark mode and light mode
* Responsive design
* Loading spinner

## Technologies Used

* HTML
* CSS
* JavaScript
* Fetch API
* Local Storage
